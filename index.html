1. Introducción

El suministro de agua es un recurso limitado y vital. Las fugas provocan desperdicio, daños materiales y costos elevados. Este proyecto desarrolla un sistema capaz de detectar fugas mediante sensores analógicos y acústicos, aplicando conceptos como:

  • Sensores analógicos
  • Instrumentación acústica
  • Acondicionamiento de señales
  • Interfaz HMI

2. Descripción del tema aplicado

Sensores analógicos: miden variaciones de presión.

Instrumentación acústica: detecta vibraciones internas.

Acondicionamiento de señal: se convierte la señal analógica en digital.

Interfaz HMI: visualización clara de datos en la web.

3. Desarrollo del proyecto

3.1 Hardware

  • Sensor de presión 174 psi: mide variaciones internas.
  • Sensor acústico industrial: registra vibraciones.
  • Arduino: convierte señales analógicas a digitales.
  • Pantalla integrada: muestra valores en tiempo real.

3.2 Método de medición

  • Instalación de sensores
  • Lectura de voltaje
  • Conversión ADC
  • Interpretación de datos
  • Clasificación del estado

4. Resultados

  • El sensor acústico detectó vibraciones anormales.
  • El sensor de presión mostró caídas bruscas.
  • La interfaz web funcionó en tiempo real.
  • El sistema reaccionó rápidamente.
  • Estructura escalable y flexible.

5. Conclusiones

Se comprobó que la detección temprana de fugas es posible usando sensores analógicos y acústicos. El sistema permite análisis preciso, monitoreo constante y puede evolucionar hacia IA o automatización.

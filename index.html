Presentación

Sistema de Detección Inteligente de Fugas de Agua

Medición e Instrumentación · 4 de diciembre de 2025

Juan Pablo Jesús Hernández Reyes · Mildred Aleshka · Arturo Yeshua · Jakob

1. Introducción

Las fugas de agua representan un problema constante que ocasiona desperdicio, daños materiales y costos elevados.

Este proyecto desarrolla un sistema que detecta fugas mediante sensores analógicos y acústicos, aplicando conceptos como:

  • Sensores analógicos
  • Instrumentación acústica
  • Acondicionamiento de señales
  • Interfaz HMI

2. Tema aplicado

Sensores analógicos: miden variaciones de presión.

Instrumentación acústica: detecta vibraciones internas.

Acondicionamiento de señal: convierte la señal para lectura digital.

Interfaz HMI: panel web para visualizar datos.

3. Desarrollo del proyecto

3.1 Hardware

  • Sensor de presión 174 psi: mide variaciones internas.
  • Sensor acústico industrial: detecta vibraciones.
  • Arduino: digitaliza las señales.
  • Pantalla integrada: muestra datos en tiempo real.

3.2 Método de medición

  • Instalación de sensores
  • Lectura de voltaje
  • Conversión ADC
  • Interpretación de datos
  • Clasificación del estado

4. Resultados

  • El sensor acústico detectó vibraciones anormales.
  • El sensor de presión mostró caídas bruscas.
  • La interfaz funcionó en tiempo real.
  • El sistema reaccionó rápidamente.
  • Diseño escalable y flexible.

5. Conclusiones

La detección temprana de fugas es posible usando sensores analógicos y acústicos. El sistema permite análisis preciso y monitoreo constante, con potencial para IA y automatización.
